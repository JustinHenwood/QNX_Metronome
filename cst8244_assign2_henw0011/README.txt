Status
The project consists of a single program named metronome.  Its purpose is emulate a metronome in the form of a QNX resource manager.  When the program is running, the user has the option of pausing, stopping, starting, setting a new tempo for the metronome and quiting out of the device driver.  This is achieved through the use of the echo command followed by the desired command and any required additionl inputs. There are no compile or runtime errors/warnings.
